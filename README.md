# leancloud-cli
> Leancloud cli for options curd.

## installation
```shell
brew install afeiship/jswork/lcc
```

## usage
```shell
# help
lcc -h

# get
lcc 60f77c8e85071346450995d3 -r

# update
lcc 60f77c8e85071346450995d3 -u -v "vvvvv-is-value"
```
